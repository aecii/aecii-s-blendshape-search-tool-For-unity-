My Blendshape Search tool adds a searchable window to Unity so you don't have to scroll through a massive blendshape list.

Type to filter — only shows blendshapes that match what you type
Sliders work as normal — changes stick, fully undoable with Ctrl+Z
Non-zero values highlight green so active ones stand out
"0" button next to any active blendshape for a quick reset
"Show non-zero only" toggle to see just what's currently in use
"Reset ALL to 0" button with a confirmation prompt
Auto-detects whichever object you click in the Hierarchy


Import the unity package. 
And then open it via Window → aecii's Blendshape Search tool.
